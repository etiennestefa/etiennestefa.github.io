Struttura della Pagina Web

Il sito è costituito da una singola pagina web statica.

File e Cartelle

index.html

Punto di accesso al sito. Porta visualizzare /index/ita.html.

index

Cartella contenente la pagina principale del sito.

ita.html

Pagina principale del sito. Il suo contenuto è diviso tra i restanti file html la cui unione viene effettuata a livello utente.

header

Cartella contenente il titolo della pagina web.

header.html

Titolo della pagina web.

menu

Cartella contenente il menù della pagia web.

menu.html

Menù della pagina web.

credits

Cartella contenente la sezione crediti della pagina web.

credits.html

Sezione crediti della pagina web.

slider

Cartella contenente lo slider della pagina web.

slider.html

Slider della pagina web.

pages

Cartella contenente le sezioni della pagina web.

contatti.html

Sezione contatti della pagina web.

documenti.html

Sezione documenti della pagina web.

gruppo.html

Sezione gruppo della pagina web.

sede.html

Sezione sede della pagina web.

images

Cartella contenente le immagini della pagina web.

files

Cartella contenente i file della pagina web.

bilanci

Cartella contenente i bilanci.

altro

Cartella contenente documenti di altra tipologia.
