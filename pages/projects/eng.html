Etienne’Emulated Didactic Machine

Etienne’s Emulated Didactic Machine is a project made in 2019 during my Informatics studies in Catania Univesity. The project briefly consisted of:

  • Realizing an Instruction Set
  • Realizing its Assemblative language
  • Making an Assembler
  • Making an Emulator of a machine able to work with the Istruction Set

The project has been developed using C++ Imperative language, Haskell Functional language and wxWidgets GUI.

This Project allowed me to obtain 30/30 cum laude in Computers Architecture Course. Learn more about EsEDM here.

Enigma!

Enigma! is a C/C++ based emulator of Germany’s Enigma machine that was used during World War II. It is Windows Console based and it allows users to:

  • Set the machine
  • Encode / Decode
  • Encode / Decode from .txt files

Enigma! , developed between 2017 and 2018, has been presented during the oral part of my High School Exams.
